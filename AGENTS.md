## Project Summary
A modern, trust-focused C2C marketplace for buying and selling used laptops and gadgets, featuring escrow-based payments, inspection periods, and transparent device history.

## Tech Stack
- Next.js 15+ (App Router, Turbopack)
- Tailwind CSS 4
- Framer Motion
- Lucide React
- Radix UI
- TypeScript

## Architecture
- `src/app/`: Page routes and layouts
- `src/components/`: Reusable UI components
- `src/lib/`: Utility functions and shared types

## User Preferences
- Clean, minimal, professional fintech-style UI
- White background with soft neutral accents
- Mobile-first responsiveness
- No crypto jargon

## Project Guidelines
- Use functional components with TypeScript
- Emphasize trust and security in copy
- Follow existing component patterns (Radix UI)

## Common Patterns
- Escrow-based transaction flow: List -> Pay -> Inspect -> Complete
- Condition grading system (e.g., Mint, Good, Fair)
